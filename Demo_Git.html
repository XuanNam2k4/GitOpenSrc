• Sử dụng terminal hoặc Cmd:
  • git config --global user.name "":Tên username của Github
  • git config --global user.email "":Tên email của Github
  • git init: Khởi tạo 1 Git Repo Local
  • git status: Kiểm tra có bất kỳ sự thay đổi nào không (kiểm tra lần 1)
  • git add (Ví dụ): git add Demo_Git.html: Cập nhật toàn bộ thay đổi lên Staging Area (khu vực chờ)
  • git status:(kiểm tra lần 2)
  • git commit -m "Nội dung": Lưu từ khu vực chờ vào local repository
  • git log: Để kiểm tra commits
  • git remote add origin link_url_repo: Gắn repo local với repo GitHub từ xa, để sau đó bạn có thể git push (đẩy code lên) hoặc git pull (kéo code về). (Có sẵn trên github)
  • git branch -M main: Tạo nhánh main trong Repo trên Github
  • git push -u origin main: Đẩy code thay đổi lên Github
  • git pull: Đồng bộ code từ Github về Local
  • git clone link_url_repo: Lấy code từ một Repo nào đó về Local
  • git branch: Xem nhánh hiện tại
  • git branch ten_branch_moi: Tạo một nhánh mới
  • git checkout ten_branch: Chuyển sang làm việc trên nhánh nào đó
  • git checkout main: Quay lại làm việc trên nhánh main
